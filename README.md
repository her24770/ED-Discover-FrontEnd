# DISCOVER FRONT-END

## Pasos para ejecutar el Proyecto

### Verificar que tienes Node.js 20 o superior

Ejecutar el Comando:
```
node --version
```

### Verificar que tienes Yarn instalado

Ejecutar el Comando:
```
yarn --version
```

### Instalar todas las dependencias
Ejecutar el Comando:

``` 
yarn install
```

### Levantar el servidor de desarrollo


Comando: 
```
yarn dev

```
